How-to Guide: Install-to-Hello World! Scala on IntelliJ

Assumptions and Notes

  • Mac
  • IntelliJ 12 Ultimate installed
  • Java 6 or higher installed

Note: This guide is just an elaboration on the one here.

0. Open IntelliJ

1. Install Plugin

File > Other Settings > Configure Plugins ...

Make sure Scala plugin is selected, hit "Finish"

2. Create Scala project with project wizard

File > New Project

Scala Module

Name your project

Scala settings: select "Set Scala Home"

"Download Scala" - select one to "Download and Install"

set path above to point to the Scala module you just downloaded/installed

select "Make global libraries" to make your life easier in the future

3. Check project libraries

File > Project Structure:

Modules should have "Scala" under your new project name

Facets should have "Scala"

4. Create object

In project view, right-click "src" folder > New > Scala class

name your class (say, HelloWorldApp), select "Object" in dropdown

object HelloWorldApp{
		def main(args: Array[String]){
		print("Hello World!");
		}

5. Run application

Ctrl + Shift + F10

Or. Right-click class file, "Run HelloWorldApp.main()"
