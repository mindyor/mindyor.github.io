Scala with IntelliJ

Install to Hello World!

Assumptions and Notes

  • Mac
  • Java 6 or higher installed
  • IntelliJ 12 Ultimate installed

Note: This guide is just an elaboration on / supplement for the guide here. Work in progress. E-mail for comments/feedback (especially "what. clarify please").

Instructions

0. Open IntelliJ

1. Install Plugin

  1. File > Other Settings > Configure Plugins ...
  2. Make sure Scala plugin is selected, hit "Finish"

2. Create Scala project with project wizard

  1. File > New Project
  2. > Scala Module
  3. Project Name: Name your project
  4. Scala settings: select "Set Scala Home"
  5. "Download Scala" - select one to "Download and Install"
  6. Set path above to point to the Scala module you just downloaded/installed
  7. Select "Make global libraries" to make your life easier in the future

3. Check project libraries

  1. File > Project Structure
  2. Modules: should have "Scala" under your new project name
  3. Facets: should have "Scala"

4. Create object

  1. In project view, right-click "src" folder > New > Scala class
  2. name your class (ex. "HelloWorldApp"), select "Object" in dropdown
    3. object HelloWorldApp{
	def main(args: Array[String]) {
		print("Hello World!");
	}
}

5. Run application

  1. Ctrl + Shift + F10
  2. Or. Right-click class file, "Run HelloWorldApp.main()"
