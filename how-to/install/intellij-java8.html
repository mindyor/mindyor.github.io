Java 8 with IntelliJ

Install to Hello World!

Assumptions and Notes

  • Mac
  • Java 6 or higher installed
  • IntelliJ 12 Ultimate installed
  • Java 8 SDK downloaded (can be found here. BE CAREFUL TO GET THE JDK NOT the JRE.)

Note: This guide is inspired by the guide here. Work in progress. E-mail for comments/feedback (especially "what. clarify please").

Instructions

0. Open IntelliJ

1. Create Java 8 project with project wizard

  1. File > New Project
  2. > Java Module
  3. Project Name: name your project
  4. Project SDK: select in dropdown (or set path above to point to the JDK dir downloaded way back in "Assumptions")
  5. "Next"
  6. "Finish"

2. Create a new class

  1. In project view, right-click "src" folder > New > Java class
  2. name your class (ex. "Java8App"), select "Object" in dropdown
    3. public Java8App{
	public static void main(String[] args) {
		System.out.println("Hello World!");
	}
}

3. Run application

  1. Ctrl + Shift + F10
  2. Or. Right-click class file, "Run Java8App.main()"
