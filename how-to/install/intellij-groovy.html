Groovy with IntelliJ

Install to Hello World!

Assumptions and Notes

  • Mac
  • Java 6 or higher installed
  • IntelliJ 12 Ultimate installed
  • Groovy SDK downloaded (can be found here.)

Note: This guide is just an elaboration on / supplement for the guide here. Work in progress. E-mail for comments/feedback (especially "what. clarify please").

Instructions

0. Open IntelliJ

1. Install Plugin

  1. File > Other Settings > Configure Plugins ...
  2. Make sure Groovy plugin is selected, hit "Finish"

2. Create Groovy project with project wizard

  1. File > New Project
  2. > Groovy Module
  3. Project Name: Name your project
  4. "Next"
  5. Please specify Groovy SDK: select in dropdown (or set path above to point to the Groovy SDK dir downloaded way back in "Assumptions")
  6. "Finish"

3. Create a new class

  1. In project view, right-click "src" folder > New > Groovy class
  2. name your class (ex. "HelloWorldApp"), select "Object" in dropdown
    3. class GroovyApp{
	void hello() {
		print("Hello World!");
	}
}

4. Create a new script

  1. In project view, right-click "src" folder > New > Groovy Script
  2. name your script (ex. "main"), select "Groovy Script" in dropdown
    3. app = new GroovyApp()
app.hello()

5. Run application

  1. Ctrl + Shift + F10
  2. Or. Right-click class file, "Run HelloWorldApp.main()"
